# 🎵 SonoVault, reproductor de música - Python + PyQt5

Aplicación de escritorio desarrollada en Python que permite reproducir música almacenada localmente, gestionar biblioteca y visualizar información enriquecida mediante integración con API externa.

---

## 🚀 Funcionalidades

- Reproducción de archivos de audio (play, pausa, siguiente, anterior)
- Selección de carpeta de música local
- Gestión de favoritos
- Búsqueda y filtrado por:
  - Artista
  - Álbum
  - Palabra clave
- Visualización de metadatos de canciones
- Integración con API de Last.fm para obtener:
  - Información de artistas
  - Portadas de álbumes
- Interfaz gráfica desarrollada con PyQt5

---

## 🛠️ Tecnologías utilizadas

- Python  
- PyQt5 (interfaz gráfica)  
- SQLite (gestión de datos local)  
- Mutagen (lectura de metadatos de archivos de audio)  
- API Last.fm  

---

## 🧠 Conceptos aplicados

- Programación orientada a objetos (POO)
- Manejo de eventos en interfaces gráficas
- Integración de APIs externas
- Procesamiento de archivos
- Persistencia de datos
- Separación de lógica y presentación (UI)
